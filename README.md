# L7Plot

🌍 Plot for L7

## 本地开发

```bash
# 安装项目依赖
$ npm install
$ npm run bootstrap

# 实时编译各 package 并启动 storybook
$ npm run dev

# 运行网站
$ npm run dev-website

# 运行单元测试
$ npm run test

# 打开 electron 运行单元测试，并监听测试文件
$ npm run test-live
```
