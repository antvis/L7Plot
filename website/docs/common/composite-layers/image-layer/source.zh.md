### `options.`source

`SourceOptions` required

数据配置，详见 [Source](/zh/docs/api/source)。

```js
{
  source: {
    'https://gw.alipayobjects.com/mdn/rms_816329/afts/img/A*4k6vT6rUsk4AAAAAAAAAAAAAARQnAQ',
    {
      parser: {
        type: 'image',
        extent: [121.168, 30.2828, 121.384, 30.4219],
      },
    },
  }
}
```
