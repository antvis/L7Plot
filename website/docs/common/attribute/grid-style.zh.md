### `options.`style

`object` optional

全局样式。

```js
{
  style: {
    coverage: 0.9,
    angle: 0,
    opacity: 1.0,
  }
}
```

#### `style.`opacity

`number` optional default: `1`

透明度。

#### `style.`coverage

`number` optional default: `0.9`

覆盖度，范围 0 到 1。

#### `style.`angle

`number` optional default: `0`

旋转角度，范围 0 到 360。
